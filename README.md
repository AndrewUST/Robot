# Robot
implementación de un sistema en ROS que permite al robot móvil realizar movimientos autónomos basados en deteccion de marcas visuales (aruco).

# Componentes

- Motor Encoder x2
- Esp32
- Raspberry pi 4
- Sensor ultrasonico
- Bateria LIPO
- Power Bank
- Camara
- Ruedas de Goma
- Rueda loca circular
